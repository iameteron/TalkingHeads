TalkingHeads

An interactive platform for studying LLM agents that can act in unfamiliar environments, ask an operator for help, and continue from grounded dialogue.

Live Demo Prompts Code Demo Video

Try the System

Live Demo

The hosted demo is available at http://158.160.220.30/play/. If the hosted demo is unavailable, use the local installation instructions.

Local Installation

Install dependencies with pip install -r requirements.txt and run play_web/scripts/play-serve.sh start.

Supported Worlds

Craftax-style worlds, Exo-Planet transfer settings, and ARC-AGI-3 human-helper interaction workflows.
